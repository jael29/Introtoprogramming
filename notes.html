Lists

Lists can be ordered(ol tag), or unordered (ul tag).

Ordered lists by default are assigned a number starting with one.

Unordered lists by default use bullet points.

You can write the lists items (li tag) without the closing tag, and the list will still know it is a listed item, and proceed to the next one.

However, you cannot leave out the ol or li closing tags, because if you do. The browser will not know when the list closes, and will include everything else as a list also.

The p tag for paragraph is another example, that you could leave out the closing tag, and the browser will still know it is a paragraph and when it closes.

Image placeholders

Sometimes when you're working on a Web page, you don't know what images you're going to use, but you still want a picture to use as a place holder.
Websites like placebear.com and placekitten.com are designed to make this super easy.
All you have to do to get an image of any given size is put the width and height of the image at the end of the URL, as in https://placebear.com/200/300.

If you enjoyed playing with those, you can find other similar sites by doing a search for "image placeholder",
and you'll find that there are gazillions more of them out there—like Lorem Picsum and FillMurray.com

Parts of an HTML document

Everything we have seen so far lives in the 2nd part of the HTML document. The body.

The body.

  1. Lists
  2. paragraphs
  3. images
  4. divs
  5. breaks

The Head

This is where we put important pieces of information about the document.

Both the head and the body occur inside a single document called HTML.

Few things that appear inside the Head of an HTML document.

  1. The Title
    • This is the only piece that is required.
      • This is placed between and opening and closing title tag.
  2. The Style (CSS)
    • Colors
    • Fonts
  3. Scripts
    • Programs that run in the browser. (javascript)
  4. Metadata
    • other information about the page itself.
    • keywords for search engines.

Things that appear inside the HTML tag itself

  1. The language of the page en = english,
    • this will be used for the webpage or screen reader to read or translate.

Examples of structure of head, body, title, doctype, and html tags.

<!DOCTYPE html> <html lang="en"> <head> <title>Title of your page </title> </head> <body> Content goes here! (Things like paragraphs, links, and images.) </body> </html> </p>
